# 📝 Spring Boot ToDo List

A simple and efficient **ToDo List Web Application** built using **Spring Boot**. This project allows users to manage daily tasks through CRUD (Create, Read, Update, Delete) operations with a clean and user-friendly interface.

---

## ✨ Features

* ➕ Add new tasks
* 📋 View all tasks
* ✏️ Update existing tasks
* ❌ Delete tasks
* 🌐 Simple and user-friendly interface

---

## 🛠️ Technologies Used

* Java
* Spring Boot
* Spring MVC
* Spring Data JPA
* Maven
* Thymeleaf
* HTML5
* CSS3
* MySQL

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/SpringBoot-ToDoList.git
```

### Navigate to the Project

```bash
cd SpringBoot-ToDoList
```

### Install Dependencies

```bash
mvn clean install
```

### Run the Application

Using Maven:

```bash
mvn spring-boot:run
```

Or using the Maven Wrapper:

```bash
./mvnw spring-boot:run
```

---

## ⚙️ Database Configuration

Update your database configuration in:

```text
src/main/resources/application.properties
```

Example:

```properties
spring.datasource.url=jdbc:mysql://localhost:3306/todolist
spring.datasource.username=root
spring.datasource.password=your_password
```

---

## 🌐 Access the Application

After starting the application, open your browser and visit:

```text
http://localhost:8080/
```
